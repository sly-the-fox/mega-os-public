# Improvement Proposals

Active queue of system improvement proposals. Reviewed during `/weekly-review`.

## Status Key
- **proposed** — awaiting user review
- **approved** — ready for implementation
- **in-progress** — specialist implementing
- **rejected** — user declined
- **verified** — implemented and confirmed positive impact
