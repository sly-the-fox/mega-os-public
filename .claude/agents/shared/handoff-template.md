# Handoff Template

## Objective
What needs to happen next.

## Relevant Context
Only the information necessary for the next agent.

## Constraints
Scope limits, approvals, deadlines, tool restrictions, quality standards.

## Definition of Done
What counts as complete.

## Risks / Notes
Known issues, blockers, uncertainties, or warnings.

## Requested Next Action
Exact action requested from the receiving agent.
